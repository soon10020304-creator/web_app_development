# 產品需求文件 (PRD) - ZenFocus 沉浸式個人專注力與白噪音工作區

---

## 1. 專案概述

### 1.1 背景與動機
在現代數位工作與學習環境中，干擾無處不在（如社群媒體、即時訊息等），這使得用戶極難進入「深度工作 (Deep Work)」或「心流 (Flow State)」狀態。現有的專注力工具大多功能單一，且介面設計生硬、缺乏視覺美感，無法在聽覺與視覺上營造出讓人沉浸的安寧環境。

**ZenFocus** 是一個專為克服分心而設計的沉浸式個人專注力與白噪音工作區。透過極簡的毛玻璃美學 (Glassmorphism)、舒適的漸變動態背景以及可客製化混音的多軌白噪音，本系統在視覺與聽覺上為用戶阻絕外界干擾。此外，系統內嵌輕量級任務看板與數據分析，提供專注與回顧的完整閉環，幫助用戶高效完成目標。

### 1.2 目標用戶
1.  **遠端工作者與程式開發者 (Remote Workers & Developers)**：需要排除家中雜音、建立工作儀式感以維持高效工作。
2.  **備考學生族群 (Students)**：希望透過番茄鐘克服拖延症，分階段高質量複習功課。
3.  **創意工作者與設計師 (Creative Professionals)**：需要純淨的背景音效來激發右腦靈感。

### 1.3 核心價值主張
*   **視覺與聽覺雙重沉浸**：藉由精心調配的暗黑美學與高質感環境音效，打造專注儀式感。
*   **無縫的任務與計時聯動**：將當前最核心的任務鎖定在計時畫面上，減少視線干擾與認知負荷。
*   **隱私與數據持久化**：使用 Flask + SQLite 輕量化後端儲存，保障數據隱私的同時，能隨時跨裝置追蹤歷史專注心流。

## 2. 功能需求

本系統規劃了 5 大主要功能，以下以使用者故事 (User Stories) 的格式詳細說明：

### 2.1 沉浸式番茄計時器 (Immersive Pomodoro Timer)
*   **使用者故事**：
    *   *作為一名學生，我希望有一個能自由設定時間且畫面精美的番茄鐘，以便我能以「專注 25 分鐘、休息 5 分鐘」的科學循環進行高效讀書。*
    *   *作為一名設計師，我希望計時器在專注和休息時能以柔和漸變的動態主題色彩切換，以便我的大腦能透過視覺直觀感知當前狀態並放鬆。*
*   **功能描述**：
    *   提供「專注」、「短暫休息」、「長休息」三種倒數模式（預設分別為 25、5、15 分鐘，支持自定義）。
    *   使用極細微光的 SVG 環形進度條流暢倒數，計時器進行時伴隨極簡微弱的粒子背景動畫。
    *   專注模式背景為深邃太空藍；休息模式背景過渡為療癒森野綠。
    *   倒數結束時播放禪意十足的磬聲 (Singing Bowl) 或風鈴聲。

### 2.2 多通道環境白噪音混音器 (Multi-channel Ambient Sound Mixer)
*   **使用者故事**：
    *   *作為一名程式開發者，我希望可以自由搭配與開關雨聲、篝火聲等不同的白噪音，並能微調各自的音量，以便我能調配出最適合我自己進入編碼心流的背景音樂。*
*   **功能描述**：
    *   內建 6 個高品質、可無縫循環的環境聲音軌：夏日細雨、林間篝火、溫柔海浪、復古咖啡廳、微風吹拂、環境鋼琴。
    *   每一軌皆有獨立的開關 (Toggle) 及音量滑桿，用戶可以靈活組合混音（例如：60% 雨聲 + 30% 篝火 = 溫馨雨夜）。
    *   提供「一鍵隨機搭配 (Surprise Me)」與「全域靜音 (Mute All)」控制。

### 2.3 專注任務與心流看板 (Focus & Flow Todo Board)
*   **使用者故事**：
    *   *作為一名遠端工作者，我希望能在開始專注前建立今日待辦清單並估算番茄數，甚至能將當前正要執行的唯一任務鎖定顯示在計時器主畫面上，以便我心無旁鶩地擊破它，而不會被其他瑣碎任務分散精力。*
*   **功能描述**：
    *   輕量級任務清單，支持新增、編輯、打勾完成、刪除任務。
    *   可設定優先級（高/中/低）與估計番茄數（以紅色番茄 🍅 顆數表示）。
    *   「專注鎖定 (Timer Focus Lock)」：點選任務鎖定後，任務文字會以半透明毛玻璃效果懸浮顯示在計時器正下方，且進入計時後自動隱藏其他清單。
    *   完成任務時播放精緻的五彩紙屑 (Confetti) 灑落動畫，給予即時正向反饋。

### 2.4 個人專注數據統計儀表板 (Focus Analytics Dashboard)
*   **使用者故事**：
    *   *作為一名注重自我成長的人，我希望能夠查看自己過去一週累積的專注總時長、番茄鐘個數與任務完成趨勢，以便調整我的工作與休息節奏。*
*   **功能描述**：
    *   統計並展示：今日專注時長、本週累積時長、完成的番茄數、任務完成率。
    *   以極簡、科技感的圖表展示專注時間趨勢（如折線圖）與專注/休息比例（如環形圖）。
    *   所有數據透過 Flask API 存入後端 SQLite 資料庫，確保數據的持久性與跨裝置儲存。

### 2.5 心流日誌與情緒溫度計 (Flow Journal & Mood Tracker)
*   **使用者故事**：
    *   *作為一個長期處於高壓環境的人，我希望在每次完成番茄鐘後可以快速記錄自己當下的專注效率與心情，以便隨時回顧自己的身心狀態與專注心流曲線。*
*   **功能描述**：
    *   番茄鐘計時結束後，自動彈出精緻的情緒溫度計小視窗，引導用戶評估「專注程度」（流暢/一般/分心）與選擇「情緒氣象」（平靜、充實、疲憊、焦慮）。
    *   提供一欄 50 字以內的心流筆記區。
    *   日誌以精美的時間軸 (Timeline) 形式呈現在儀表板下方，形成個人的「專注日誌本」。

## 3. 非功能性需求

### 3.1 技術限制與架構
*   **後端框架**：使用 Python 3 + Flask 框架。
*   **前端模板**：使用 Jinja2 模板引擎進行 HTML5 頁面渲染。
*   **資料庫**：使用 SQLite3 儲存用戶的所有任務清單、專注日誌、番茄紀錄以及自訂設定，架構極簡且無需部署大型數據庫。
*   **前端樣式**：使用 Vanilla CSS3 (原生 CSS)，手動編寫細緻的毛玻璃 (Glassmorphism) 與霓虹微光變色效果，避免使用過重的 TailwindCSS 以確保系統的純淨度與流暢動畫。

### 3.2 效能與安全考量
*   **響應式佈局 (Responsive Layout)**：支援 PC、平板與手機端。在行動裝置上，混音器與任務清單會優雅地摺疊為側邊/底部抽屜。
*   **音頻流暢度**：白噪音採用無縫循環播放技術，確保聲音過渡不卡頓、不中斷。
*   **單一使用者儲存**：在無登入機制的 MVP 版本中，資料庫採用 local-db 機制（或依據 session ID 隔離），保障每個用戶看到的都是自己專屬的專注空間。

## 4. MVP 範圍 (Minimum Viable Product)

| 優先權 | 功能模組 | 具體功能描述 |
| :--- | :--- | :--- |
| **Must Have (必須有)** | 沉浸式番茄計時器 | 基本倒數計時、專注/休息切換、計時結束聲音提示、環形進度條。 |
| | 多通道白噪音混音器 | 至少 3 軌環境音（雨聲、篝火、咖啡廳），獨立開關與音量調節。 |
| | 專注任務清單 | 任務新增、打勾完成、刪除，以及基本的 Local/Session 資料庫儲存。 |
| **Should Have (應該有)** | 專注鎖定 (Timer Focus Lock) | 可將特定任務鎖定懸浮在番茄鐘下方，進入計時自動隱藏其餘部分。 |
| | 數據庫持久化與統計 | 使用 SQLite3 儲存數據，並能在儀表板顯示當日專注總時長與番茄個數。 |
| | 任務完成動畫 (Confetti) | 任務完成時的前端五彩紙屑特效。 |
| **Nice to Have (可以有)** | 多軌混音與隨機搭配 | 擴展至 6 個音軌，支持「隨機搭配 (Surprise Me)」與「全域靜音 (Mute All)」。 |
| | 心流日誌與情緒溫度計 | 計時結束彈窗自評，並以時間軸 (Timeline) 形式儲存與展現歷史日誌。 |
| | 動態環境主題背景 | 專注與休息模式下背景漸變色彩過渡，伴隨微粒子粒子特效。 |
